---
title: Installation
description: Install the Meridian TypeScript SDK.
---

```bash
npm install meridian-sdk
# or
pnpm add meridian-sdk
# or
bun add meridian-sdk
```

## Requirements

- TypeScript 5.x with `strict: true`
- Node 18+, Bun 1+, or any modern browser
- A running Meridian server (see [Docker](/self-hosting/docker))

## Examples

**Collaborative workspace** — ORSet tasks, PNCounter votes, LWWRegister title, Presence (vanilla TypeScript):

```bash
cd examples/collaborative-workspace
bun install && bun dev
```

**Live cursors** — real-time cursor tracking with `usePresence` (React):

```bash
cd examples/cursors
bun install && bun dev
```
