title: Installation
description: Install the Meridian SDK — TypeScript, Python, or Rust.
---

## TypeScript

```bash
npm install meridian-sdk
# or
pnpm add meridian-sdk
# or
bun add meridian-sdk
```

**Requirements**: TypeScript 5.x with `strict: true`, Node 18+, Bun 1+, or any modern browser.

## Python

```bash
pip install meridian-crdt

# With AES-GCM-256 encryption + Ed25519 signing:
pip install "meridian-crdt[crypto]"
```

**Requirements**: Python 3.10+, asyncio.

## Rust

```toml
[dependencies]
meridian-client = "0.1"
tokio = { version = "1", features = ["rt-multi-thread", "macros"] }
```

See [Rust SDK](/sdk/rust) for usage.

---

A running Meridian server is required for all SDKs — see [Docker](/self-hosting/docker).

## Examples

**Collaborative workspace** — ORSet tasks, PNCounter votes, LWWRegister title, Presence (vanilla TypeScript):

```bash
cd examples/collaborative-workspace
bun install && bun dev
```

**Live cursors** — real-time cursor tracking with `usePresence` (React):

```bash
cd examples/cursors
bun install && bun dev
```

**Collaborative AI agents** — four examples in one: multi-agent document summarization, Claude tool use over CRDTs, persistent agent memory across runs, and a two-agent debate. Live React dashboard with tabs for each scenario (`useCRDTMap`, `usePresence`, `useGCounter`, `useORSet`, `useLwwRegister`):

```bash
cd examples/ai-agents
bun install

# Terminal 1 — UI (http://localhost:5175)
cd ui && bun run dev

# Terminal 2 — pick an example
MERIDIAN_ADMIN_TOKEN=xxx ANTHROPIC_API_KEY=xxx bun run agents   # multi-agent coordinator
MERIDIAN_ADMIN_TOKEN=xxx ANTHROPIC_API_KEY=xxx bun run tool-use # single agent + tool use
MERIDIAN_ADMIN_TOKEN=xxx ANTHROPIC_API_KEY=xxx bun run memory   # persistent memory
MERIDIAN_ADMIN_TOKEN=xxx ANTHROPIC_API_KEY=xxx bun run debate   # two-agent debate
```

See [`examples/ai-agents/README.md`](https://github.com/Chahine-tech/meridian/tree/main/examples/ai-agents) for full setup instructions.
