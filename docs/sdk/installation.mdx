---
title: Installation
description: Install the Meridian TypeScript SDK.
---

```bash
npm install meridian-sdk effect msgpackr
# or
pnpm add meridian-sdk effect msgpackr
# or
bun add meridian-sdk effect msgpackr
```

`effect` and `msgpackr` are peer dependencies.

## Requirements

- TypeScript 5.x with `strict: true`
- Node 18+, Bun 1+, or any modern browser
- A running Meridian server (see [Docker](/self-hosting/docker))

## Example

A full working example (ORSet tasks, PNCounter votes, LWWRegister title, Presence) is available in the repository:

```
examples/collaborative-workspace/
```

```bash
cd examples/collaborative-workspace
npm install && npm run dev
# or
pnpm install && pnpm dev
# or
bun install && bun dev
```
