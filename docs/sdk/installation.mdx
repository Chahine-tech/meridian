---
title: Installation
description: Install the Meridian TypeScript SDK.
---

```bash
bun add @meridian/sdk effect msgpackr
# or
npm install @meridian/sdk effect msgpackr
```

`effect` and `msgpackr` are peer dependencies.

## Requirements

- TypeScript 5.x with `strict: true`
- Node 18+, Bun 1+, or any modern browser
- A running Meridian server (see [Docker](/self-hosting/docker))
